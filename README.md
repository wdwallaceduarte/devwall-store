# DevWall Store

Uma landing page moderna e animada para apresentação de produtos tecnológicos, desenvolvida com HTML, CSS e JavaScript puro.

## 📸 Preview

O projeto apresenta um carrossel animado com produtos tecnológicos, incluindo:

- Apple Watch Series 10
- Air Pods Max
- Vision Pro

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript Vanilla

## ✨ Funcionalidades

- Carrossel automático de produtos
- Navegação manual com botões
- Indicadores dinâmicos
- Animações suaves com CSS
- Layout moderno e responsivo
- Efeitos visuais futuristas
